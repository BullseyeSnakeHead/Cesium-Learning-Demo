此项目采用Vanilla框架，纯前端项目。
src中有多个main.js文件，项目js相关的demo在main9.copy.js中，html中已经对main9.copy.js的canvans画布绑定，应当可以直接运行。
src——main9 copy.js为 “成都智慧地铁管理项目”的js源码。其余js文件为自学使用。
public文件夹中存放地铁图片，地铁模型，dem高程切片数据，区县矢量json等。
本项目调用和风天气API，实时显示当前成都的天气状况，温度。后期会据此添加新的canvas画布实现天气效果。
和风天气api高频调用会收费，禁止用源码的 Key/Token 在本项目外使用。
